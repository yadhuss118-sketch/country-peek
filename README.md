# CountryPeek

A React app to explore countries using the RestCountries API.

## Live Demo
https://github.com/yadhuss118-sketch/country-peek.git

## Features
- Search countries
- Filter by region
- Sort results
- Country detail page
- Dark/light mode
- Favourites (localStorage)

## Tech Stack
React, Vite, React Router

## Run Locally
npm install
npm run dev
